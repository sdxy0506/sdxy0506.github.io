android studio mac 快捷键 总结 (不断更新)

最近公司的项目都已经迁移到了Android Studio上面进行开发，刚转过来各种不适应，甚至连快捷键都不知道，现在用得顺手一些了，总结一下开发中用到的快捷键，以后会根据新用到的快捷键进行更新~

svn

描述 快捷键
提交修改 Cmd + K
更新代码 Cmd + T
VCS option Ctrl + V
比较不同 Cmd + D

coding

描述 快捷键
打开类 Cmd + O
在类中查找方法 Cmd + F12
全部文件查找 shift 双击
删除当前行 Cmd + delete
格式化代码 Cmd + option
上下移动代码 Option + Shift + Up/Down
快捷生成结构体 Cmd + Option + T
快捷覆写方法 Ctrl + O
快捷定位到行首/尾 Cmd + Left/Right
折叠展开代码块 Cmd + Plus,Minus
大小写转换 Cmd + Shift + U
查找调用的位置 Ctrl + Option + H
Find usage Alt + F7
重命名 Shift + F6
重构类结构 Cmd + F6
提取方法 Alt + Cmd + M
